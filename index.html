What is HTML?

1.HTML stands for HyperText Markup Language, used to create the structure of web pages.

2.It is the standard language used to create and design web pages.

3.HTML uses a series of elements or tags to structure content (e.g., headings, paragraphs, links).

4.HTML focuses on structure, it is often used alongside CSS for styling and JavaScript for functionality.

What is CSS?

1.CSS stands for Cascading Style Sheets, used for styling the appearance of web pages.

2.CSS supports responsive design, allowing websites to adapt to different screen sizes using media queries.

3.CSS separates content (HTML) from presentation, improving maintainability.

What is JavaScript?

1.JavaScript is a programming language that adds interactivity to web pages.

2.JavaScript is a high-level, dynamic programming language primarily used for web development.

3.It can also be used on the server side (e.g., Node.js) for building scalable applications.
